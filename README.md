# Trabajo Final - Análisis de Datos con NumPy, Pandas y Matplotlib

## Descripción
Este proyecto corresponde al trabajo final del curso, en el cual se aplican los conocimientos de **NumPy**, **Pandas** y **Matplotlib** para analizar un conjunto de datos de ventas.

El programa realiza:

- Creación y lectura de un dataset en formato CSV
- Manipulación de datos con Pandas
- Cálculos estadísticos
- Filtros de información
- Conversión de datos a NumPy
- Visualización mediante gráficos
- Análisis de resultados

---

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Visual Studio Code
